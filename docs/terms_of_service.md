# 利用規約

### 【利用規約への同意】

下記特記事項は、yobimori という名称で、漁師等の海難事故発生時に、本デバイス（【デバイスの権利関係】第 1 項に定義）を起動すると、当該本デバイスの位置情報をアプリ利用者に対して、通知するサービス（以下「本サービス」といいます）の利用規約（以下「本利用規約」といいます）を定めるものです。申込者は、本利用規約を全文お読みいただいた上で、本利用規約への同意をお願いいたします。

### 【本サービスの内容】

1. 本規約により提供する「本サービス」の内容は次のとおりです。   
     ① デバイス等の貸与  
     ② 事故発生時の自動通報または手動通報  
     ③ 事故発生時の通報の受信  
     ④nanofreaks の算定基準により算出された、救助者が負担した費用の事故者への請求  
     ※救助者とは：「本サービス」による自動通報または手動通報を受信し、事故者の救助活動を行った者  
     ※事故者とは：海難事故発生時に「本サービス」にて、自動通報または手動通報を行ったものまたはその使者
2. 「yobimoriβ 版」は開発中のサービスであり、今後の正式版サービスのリリースを目的とした実験的サービスとして、当社が定める条件で契約者等に提供されるものです
3. 当社は、「yobimoriβ 版」の仕様、サービスの名称、内容、機能及びデザイン等を契約者に対して、予告なく変更することがあります。なお、この当社の措置につき、契約者は異議を申し立てることはできず、当該措置により契約者に何らかの損害が生じたとしても、当社は一切の責任を負いません。
4. 「yobimoriβ 版」の性質上、「yobimoriβ 版」は、バグや瑕疵、誤動作など、正常に稼働できない症状等の不具合を、含み得るものとして提供されます。当社は、「yobimoriβ 版」の正確性、継続性、完全性及び確実性を含む信頼性、有用性、可用性、利用可能性、セキュリティ保護性、無エラー性、無ウイルス性、品質満足度、並びに契約者の特定目的への適合性を含む、いかなる種類の保証も行わず、これらに起因して、契約者等及びその他の第三者が被った損害に関し、一切の責任を負いません。例えば、「yobimoriβ 版」の誤作動による燃料費の損失、動作不良による事故など。また、本製品は海上での利用を想定しており、その他の場所での利用はできず、海上以外で利用した事に起因して、契約者等及びその他の第三者が被った損害に関し、一切の責任を負いません。
5. 契約者コンテンツ等の「yobimoriβ 版」を利用して、作成した契約者の情報は、予告なく全部又は一部が削除又は変更される場合があります。また、当社は、「yobimoriβ 版」運用終了後に提供される正式サービスにおいて、当該情報を引き継ぐ義務を負わないものとします。
6. 当社は、契約者に対して、「yobimoriβ 版」を補修する義務及び「yobimoriβ 版」のアップデート版を提供する義務を負いません。ただし、当社の判断により、契約者に対して「yobimoriβ 版」のアップデート版を提供する場合があります。また、当社は、当該アップデートによって、当該アップデート前の「yobimoriβ 版」の機能及び性能が維持されることを保証するものではありません。

### 【デバイスの権利関係】

2. 「デバイス利用者」とは、本デバイスを利用する者を意味します。
3. 「アプリ利用者」とは、漁師等の海難事故発生時に、デバイスから発信された位置情報を、受信することのできるアプリケーション（以下「本アプリ」といいます）を利用する者を意味します。
4. 当社は、本申込書に基づき、申込者に対して本デバイスを下記有効期間中賃貸するものとします（以下、当該契約を「本契約」といいます）。申込者は、本サービス利用中は本利用料を支払うとともに、下記有効期間終了後は本デバイスを当社に対して、返却するものとします。
5. 当社は、申込者に対し、本デバイスを本申込書に定める個数分を送付するものとし、送付費用は当社が負担するものとします。但し、別途アプリ利用者が本アプリを通じて、本デバイスの送付先を指定した場合は、当該指定された送付先とします。なお、発送時点以降における配送遅延、誤配送、紛失その他の事由により生じた損害は、申込者が負担するものとします。
6. 申込者は、当社が適切と認める資格を満たす方をデバイス利用者として、本デバイスを利用させることができるものとします。但し、申込者はデバイス利用者に対して、本利用規約と同様の義務を遵守させるものとし、当該デバイス利用者による当該利用規約の違反は、申込者の違反とみなすものとします。
7. 本デバイスの所有権は当社にあり、デバイス利用者に本利用規約に従って利用させる場合を除き、申込者は当社の事前の書面による同意なくして、第三者に対し本デバイスの譲渡、転貸又は担保提供、その他の処分を行ってはならないものとします。

9. 前項の規定にかかわらず、本デバイスについて損傷、故障又は紛失その他の事由が生じた場合、申込者は、自らの責任と費用負担において、直ちに当社に連絡するものとします。この場合、当社が別途定める料金を請求いたします。

### 【対価の支払】

1. 申込者は、本サービスを利用する場合には、当社規定の利用料金 （以下 「本利用料」 といいます） を口座振替により支払うものとします。ただし、口座振替開始までにかかるまでの期間の料金とデバイス導入費を、共に初期費用として、当社指定の口座に振込にて支払っていただきます。
2. 本利用料は、本サービスの Web サイト内に記載するものとします。
3. 当社は、乙の了承を得ることなく、本利用料を変更することができるものとします。
4. 本利用料の変更は、変更後の利用料が本サービスのサイトまたは本サービス内に表示された時点より、効力が生じるものとします。
5. 本デバイスから通信基地局までの通信に係る費用については、本利用料に含まれますが、本サービスの利用にともない、発生するその他諸費用（アプリ利用時のスマートフォン通信料金及びスマートフォンの月額基本料金等を含みますが、これに限られません） については、申込者の負担とします。
6. 振込⼿数料は申込者のご負担となります。
7. 初期費用は申込日より、一週間以内に支払わなくてはいけません。
8. 月額費用は毎月末払いでのご請求となります。なお、契約の開始又は終了が月の途中となる場合であっても、日割り計算はいたしません。
9. 事故者となった場合の救助費用については、事故者自身の負担とします。
10. その他の支払に関する事項は、別途定める当社規定に従うものとします。
